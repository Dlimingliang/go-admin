#### 接口返回

API接口的响应为JSON格式，并且会响应不同的HTTP状态码

1. HTTP状态码

- 200 成功
- 400 客户端参数错误 比如json格式错误
- 401 未认证 未提供认证数据或认证数据无效
- 403 未授权 无权限
- 409 冲突、业务异常
- 500 系统未知异常

2. API返回结果

```
{
    "code": 300,
    "msg": "message",
    "validationMsg : {"参数校验错误信息"}
    "data": {"数据"}
}
```
